## Exact Top.gg description

Use the following as the Top.gg long description. Keep the first paragraph at the very top.

```markdown
Manhwa Updates is a Discord bot for automatic manhwa chapter notifications. Track your favourite series, ping server roles, or subscribe to DM alerts when new chapters release.

No self-hosting required. Invite the hosted bot and use it out of the box.

Manhwa Updates Bot helps Discord communities track manhwa, manga, and manhua releases from supported websites. Add the bot to your server, choose a notification channel, track a series, and receive alerts whenever a new chapter is detected.

Perfect for manhwa servers, scanlation communities, reading groups, and anyone who wants chapter update notifications without manually checking websites.

## Features

- Automatic manhwa chapter update notifications in Discord
- Track series in server channels
- Ping roles when new chapters release
- Subscribe to personal DM alerts
- Bookmark series and update reading progress
- Search series, view info, and list chapters
- Browse supported sources with `/supported_websites`
- Slash commands for easy server setup

## Quick Start

1. Invite Manhwa Updates Bot to your Discord server.
2. Run `/help` to see the main commands.
3. Run `/settings` to choose your notification channel.
4. Run `/track new` to track a manhwa, manga, or manhua series.
5. Run `/supported_websites` to view supported sources.

## Common Commands

- `/track new` - Track a series and optionally ping a role
- `/track list` - View tracked series in your server
- `/subscribe new` - Get personal DM chapter alerts
- `/bookmark new` - Save a series and reading progress
- `/search` - Search for a title
- `/info` - View series information
- `/chapters` - Browse chapters
- `/supported_websites` - View supported websites
- `/stats` - View bot stats

## Who should use it?

Manhwa Updates Bot is built for manhwa Discord servers, manga/manhua communities, reading groups, and users who want chapter notifications from supported websites instead of manually checking each source.

If you searched for "manwha updates bot" or "manwha discord bot," you probably mean "manhwa updates bot" — this bot tracks manhwa chapter releases and sends Discord notifications.

[Support Server](https://discord.gg/TYkw8VBZkr) | [GitHub](https://github.com/MooshiMochi/ManhwaUpdatesBot)
```
